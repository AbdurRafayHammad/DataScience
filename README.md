

 **Dataset Exploration & Visualization**
Objective

The goal of this project is to explore and visualize the Iris dataset using Python. It focuses on understanding the dataset structure, summarizing key statistics, and analyzing relationships between variables.


The Iris dataset contains:

Sepal Length
Sepal Width
Petal Length
Petal Width
Species (Setosa, Versicolor, Virginica)
Tools & Libraries:
Python
Pandas
Matplotlib
Seaborn
Steps Performed:
1. Data Loading
Loaded dataset using seaborn
Displayed first rows using .head()
2. Data Inspection
Checked dataset structure using .info()
Summary statistics using .describe()
3. Data Visualization
Scatter Plot (sepal length vs width)
Histogram (petal length)
Box Plot (sepal length)
Conclusion:
Learned dataset structure and patterns
Performed basic EDA
Visualized relationships effectively
** Credit Risk Prediction**
Objective

Predict whether a loan applicant will be approved or default using machine learning.

Dataset:
https://www.kaggle.com/datasets/architsharma01/loan-approval-prediction-dataset
Features include:
Income
Loan Amount
Education
CIBIL Score
Asset Values
Loan Status (Target)
Tools & Libraries:
Python
Pandas
Seaborn
Matplotlib
Scikit-learn 
Steps Performed:
1. Data Cleaning
Removed extra spaces
Handled missing values
2. EDA
Histogram (loan amount)
Scatter plot (income vs loan)
Count plot (education vs loan status)
3. Model Training
Logistic Regression
Train-test split
4. Evaluation
Accuracy: ~79.6%


Results:
Good prediction performance
